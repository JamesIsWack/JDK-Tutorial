Tutorial on how to install JDK on non-Administrator devices.

First, download the x64 Windows compressed archive from this link

Next, extract the zip in File Explorer or 7zip. Take the JDK folder that contains the bin, and conf folders (there should be other folders as well). Take the path were the bin folder is located, and copy it using CTRL+C. After you do this, go to the Search bar on the taskbar, and type Control Panel. You need to open up the first option, and then go to User Accounts > User Accounts again > Change my system enviorment variables. Add a new variable with the variable name JAVA_HOME

Give the new variable a value, and put in the path for the JDK (the path you copied earlier). As an example, I have JDK on the root of my C drive, so the variable value is "C:/jdk18/bin" without the quots.

Thats really all there is too it, however I do reccomend using VS Code as the IDE, but NetBeans is the one provided with the course.
